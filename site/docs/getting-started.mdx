kxs-ts-gen is published to [Maven Central](https://search.maven.org/search?q=g:dev.adamko.kxstsgen).

Be sure to apply the [Kotlinx Serialization](https://github.com/Kotlin/kotlinx.serialization) plugin.

### Adding a dependency

<!-- [//]: # (@formatter:off) -->

  values={[
    { label: 'Gradle Kotlin DSL', value: 'kts', },

<TabItem value="kts">

```kts
plugins {
  kotlin("jvm") version "1.8.0"
  kotlin("plugin.serialization") version "1.8.0"
}

dependencies {
  implementation("dev.adamko.kxstsgen:kxs-ts-gen-core:0.2.1")
}
```

</TabItem>
<TabItem value="groovy">

```groovy
plugins {
  id "org.jetbrains.kotlin.jvm" version "1.8.0"
  id "org.jetbrains.kotlin.plugin.serialization" version "1.8.0"
}

dependencies {
  implementation("dev.adamko.kxstsgen:kxs-ts-gen-core:0.2.1")
}
```

</TabItem>
<TabItem value="maven">

:::info

Maven must postfix the module name with the target variant, for example: `-jvm`

:::

```xml
<dependency>
  <groupId>dev.adamko.kxstsgen</groupId>
  <artifactId>kxs-ts-gen-core-jvm</artifactId>
  <version>0.2.1</version>
</dependency>
```

</TabItem>
</Tabs>
